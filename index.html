Regular Noise

Each pixel is a random color.
var c = document.getElementById('mycanvas'); noise.noise(c);

First, we are going to start with a small patch.

It's the same as the previous one, but we are just using a small 20x20 area.
var c = document.getElementById('mycanvas'); noise.noise(c,{w:20,h:20});

Regular Noise,but progressively bigger

We are using a 40x40. In the next slide we will keep on going and overlap all the results
var c = document.getElementById('mycanvas'); noise.noise(c,{w:40,h:40});

Now we overlap these

var c = document.getElementById('mycanvas'); noise.noise(c,{w:320,h:320,border:true}); noise.noise(c,{w:160,h:160,border:true}); noise.noise(c,{w:80,h:80,border:true}); noise.noise(c,{w:40,h:40,border:true}); noise.noise(c,{w:20,h:20,border:true});

Now we make the smaller ones progressively lighter

The next slide we upscale all the smaller ones to match the largest
var c = document.getElementById('mycanvas'); noise.noise(c,{w:320,h:320,border:true}); noise.noise(c,{w:160,h:160,border:true,alpha:0.8}); noise.noise(c,{w:80,h:80,border:true,alpha:0.6}); noise.noise(c,{w:40,h:40,border:true,alpha:0.4}); noise.noise(c,{w:20,h:20,border:true,alpha:0.2});

Each smaller one is now upscaled to the largest size

var c = document.getElementById('mycanvas'); noise.noise(c,{w:320,h:320}); noise.noise(c,{w:160,h:160,fillwidth:320,fillheight:320,alpha:0.8}); noise.noise(c,{w:80,h:80,fillwidth:320,fillheight:320,alpha:0.6}); noise.noise(c,{w:40,h:40,fillwidth:320,fillheight:320,alpha:0.4}); noise.noise(c,{w:20,h:20,fillwidth:320,fillheight:320,alpha:0.2});

Now we adjust the blending mode to lighten and global alpha for each different size

var c = document.getElementById('mycanvas'); noise.octaveNoise(c,{octaves:8});

Smoothing

Here is the before and after. In the next page we will see that we can use smoothing on the previous upscale example
var c = document.getElementById('mycanvas'); noise.noise(c,{w:40,h:40,fillwidth:c.width,fillheight:c.height}); noise.noise(c2,{w:40,h:40,fillwidth:c.width,fillheight:c.height,isSmooth:true});

We can now use smoothing if we'd like.

var c = document.getElementById('mycanvas'); noise.octaveNoise(rc,{octaves:8,isSmooth:false}); noise.octaveNoise(c,{octaves:8,isSmooth:true});

Pretty cool?
